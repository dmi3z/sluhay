Пример полного расчёта с учётом коэффициентов

Пример:
Фонд выплаты артистам составляет в месяц: 1000 бел. руб.
Общее количество песен в ротации: 2000
Из них:
100 аудио песен X коэффициент «1» = 100 баллов.
95 видео песен X коэффициент «2» =190 баллов.
5 песен эксклюзивно для Sluhay X коэффициент «10» = 50 баллов.
ИТОГО: 340 баллов
Песен в ротации у определённого артиста: 10
Из них:
8 аудио песен X коэффициент «1» = 8 баллов.
1 видео песня X коэффициент «2» = 2 балла.
1 песня эксклюзивно для Sluhay X коэффициент «10» = 10 баллов.
ИТОГО: 20 баллов
Песен в ротации у определённого артиста: 10
Из них:
8 аудио песен X коэффициент «1» = 8 баллов.
1 видео песня X коэффициент «2» = 2 балла.
1 песня эксклюзивно для Sluhay X коэффициент «10» = 10 баллов.
ИТОГО: 20 баллов
Доля ариста в общем фонде вознаграждения составляет:
5,88% (20 баллов / 340 баллов)
Вознаграждение артиста за месяц составит:
1000 бел. руб. X 5,88% = 58,8 бел. руб.
